WHAT WAS BUILT
================================================================================

A native iOS app in SwiftUI that displays the comprehensive 4-week AK peptide 
protocol with:

✅ 4 weeks of complete protocol data (Weeks 1-4)
✅ Day-by-day selector (Sun-Sat)
✅ 30+ compounds across all categories
✅ Time-based sections (Waking, AM, Mid-Day, Evening, Sleep)
✅ Dynamic dosing (Vyvanse, NAD+, Wellbutrin vary by week/day)
✅ Automatic draw volume calculations (handles µg, mg, IU)
✅ Master configuration dialog (adjust all compounds)
✅ Individual compound adjustment dialogs
✅ UserDefaults persistence (saves custom doses/concentrations)
✅ Collapsible time sections
✅ Clean, native iOS UI

KEY FEATURES
================================================================================

WEEK MANAGEMENT
- 4 complete weeks of protocol data from markdown
- Proper dosing variations across weeks
- Wellbutrin taper (300mg → 225mg → 150mg)
- MOTS-C starts Week 2
- Compound variations by week

DAY MANAGEMENT
- Sun-Sat day selector
- Vyvanse: 30mg (Sun/Tue/Thu/Sat), 60mg (Mon/Wed/Fri)
- NAD+: 200mg/150mg variations
- Mon/Wed/Fri compounds (hCG, BPC-157, TB-500, GHK-Cu)
- Sun/Tue/Thu/Sat compounds (Kisspeptin, BPC-157, TB-500, GHK-Cu)
- Adamax Mon/Thu only (replaces Semax)

TIME ORGANIZATION
- Waking (🌅): VIP, hGH, Kisspeptin, hCG, Vyvanse, etc.
- AM (🌞): NAD+, MOTS-C, Retatrutide, AOD-9604
- Mid-Day (🧠): P21, N-Acetyl Semax, Adamax
- Evening (🌙): BPC-157, TB-500, GHK-Cu, KPV
- Sleep (😴): SS-31, DSIP, Pinealon

COMPOUND CATEGORIES (30+ total)
- Medications: Vyvanse, Wellbutrin, Enclomiphene
- HPTA: hCG, Kisspeptin-10
- Metabolic: Retatrutide, AOD-9604, MOTS-C
- Support: VIP, SS-31, NAD+
- Reprogramming: N-Acetyl Semax, Adamax, N-Acetyl Selank, P21
- Healing: BPC-157 (L/R), TB-500 (L/R), GHK-Cu (L/R), KPV, hGH
- Rest: DSIP, Pinealon

CONFIGURATION
- Master config: Adjust all compounds at once
- Individual adjust: Per-compound configuration
- Persistence: UserDefaults saves all customizations
- Reset: Each compound can be reset to defaults

CALCULATIONS
- Automatic draw volume calculation
- Handles µg → mg conversions
- IU handling for hGH/hCG
- Oral compounds display as "oral"
- Precision formatting (0.001-10+ mL)

================================================================================
HOW TO GET STARTED
================================================================================

OPTION 1: Quick Start (5 minutes)
1. Open Xcode
2. File → New → Project → iOS App
3. Name: AKProtocol, Interface: SwiftUI
4. Add all source files from AKProtocol/ directory
5. Run!

See QUICKSTART.md for step-by-step guide.

OPTION 2: Automated (requires xcodegen)
1. cd ios-ak-beta
2. ./create-xcode-project.sh
3. open AKProtocol.xcodeproj

See SETUP.md for detailed manual setup.
